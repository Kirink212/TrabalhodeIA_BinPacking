# TrabalhodeIA_BinPacking

Antes de tudo, para executar este trabalho precisa-se instalar a última
versão do python, permitindo assim que executemos o código principal "main.py".
Após isso, basta seguir o passo a passo abaixo:

1) Abrir a linha de comandos e mudar o diretório atual para este;
2) Execute o código main.py, através do comando "python3 main.py";
3) Perceba que as instâncias são executadas de forma seguida. Para
ver uma por uma, basta entrar no código e comentar as instâncias ou
funções indesejadas.

Além disso, todos os arquivos de saída estão armazenados nas pastas:

1) Instances-SubSet-Falkenauer-FirstFit-Solution: Armazena os resultados da 
heurística First Fit;
2) Instances-SubSet-Falkenauer-HillClimbing-Solution: Armazena os resultados da 
busca local Hill Climbing, a qual utiliza o método Random Restart.

OBS.: Os arquivos "first_fit_matrix.py" e "tentativa_simulated_annealig.py" não
devem ser executados, pois são códigos referentes a tentativas de implementação,
algoritmos que não deram certo, dentre outros.
